## Question : 

### You are given a preorder traversal of binary search tree print the post order traversal of it.

### Input Description:
You are given with preorder traversal of binary search tree

### Output Description:
Print the post order traversal of tree

Sample Input : <br/>
1 0 2 <br/>
Sample Output : <br/>
0 2 1
